# FAQ (Public-Safe)

What is CORE PACT?
- A framework and approach for multi-agent/model collaboration with safety, reliability, and clarity as core principles.

What’s in this repo?
- High-level docs, examples, and diagrams intended for a public audience. No proprietary internals or credentials.

Can I deploy CORE PACT from this repo?
- No. This is not a deployment or product repository. It’s a public, high-level showcase.

Where are the algorithms and internal prompts?
- Not included by design. This repo avoids implementation-level details to protect IP and safety.

How can I contribute?
- Propose edits to docs, improve examples and diagrams, suggest use cases, or file issues—see `CONTRIBUTING.md`.

Is there a public roadmap?
- Yes—see `ROADMAP_PUBLIC.md`. It’s intentionally sanitized and non-committal.
